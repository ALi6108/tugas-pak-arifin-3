# tugas-pak-arifin-3
data e commerce analytics
Berikut jawaban singkat berdasarkan analisis e-commerce kamu:

**1. Business Question**

* Siapa pelanggan terbaik (RFM)?
* Kategori produk mana paling efisien?
* Apakah Ad Budget berpengaruh pada penjualan?

**2. Data Wrangling**

* Mengubah format tanggal
* Rename kolom agar konsisten
* Menambahkan kolom Ad_Budget
* Mengelompokkan data (groupby) untuk analisis RFM & kategori
* Menghapus/menangani duplikasi otomatis lewat agregasi

**3. Insights**

* Bar chart menunjukkan kategori dengan efisiensi tertinggi menghasilkan penjualan besar dengan budget kecil
* Heatmap RFM menunjukkan pelanggan “Champions” memberi kontribusi terbesar
* Ad Budget cenderung berpengaruh terhadap kenaikan penjualan (terlihat dari tren/regresi)

**4. Recommendation**

* Fokus retensi pelanggan “Champions” dan “Loyal”
* Tingkatkan budget pada kategori paling efisien
* Evaluasi produk mahal dengan penjualan rendah
* Optimalkan strategi iklan berdasarkan performa data
